<all-notes> ​

Usage ​

  1. If you haven't already, go through the setup for the module.

  2. Import the <all-notes> element somewhere in the javascript side of your web-app like this:

js
import '@darksoil-studio/notes/dist/elements/all-notes.js'
  1. Use it in the html side of your web-app like this:
html
<all-notes>
</all-notes>

WARNING

Like all the elements in this module, <all-notes> needs to be placed inside an initialized <notes-context>.
